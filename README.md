# YUTA INDEX

ゆた（[@yutayuta_8e](https://x.com/yutayuta_8e)）の作品とSNSをまとめた公式リンクページです。

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run build
npm test
npm run lint
```

The featured “ゆたの世界” card links to the standalone portfolio at
<https://yutayuta-world.pages.dev/>.
